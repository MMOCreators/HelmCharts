# How to use it as a helm repo

```bash
$ helm repo add mmo 'https://MMOCreators.github.io/HelmCharts'
$ helm repo update
$ helm search mmo
NAME            	VERSION	DESCRIPTION
mmo/example     0.0.1  	A Example Helm chart in Kubernetes
```
